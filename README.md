# 🏨 Luxury Hotel Booking Website

A modern, elegant, and fully responsive hotel booking website built with **Next.js**, **React.js**, and **Bootstrap 5**. The project delivers a premium user experience with smooth animations, responsive layouts, and a clean component-based architecture, making it an excellent foundation for future backend integration.

---

## ✨ Features

- 🏨 Modern Luxury Hotel Landing Page
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Next.js
- 🎯 Interactive Hero Section
- 🛏️ Featured Rooms Showcase
- 🍽️ Restaurant & Bar Menu
- 📅 Booking Form UI
- 💬 Customer Testimonials
- 📰 Blog Section
- 📈 Animated Statistics (CounterUp)
- 🎬 Smooth Scroll Animations (AOS)
- 🎨 Modern Icons using React Icons
- ♻️ Reusable Component-Based Architecture
- 🚀 SEO-Friendly Structure

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/mz-devtech/luxury-hotel-booking.git
```

## Navigate to the Project

```bash
cd luxury-hotel-booking
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
